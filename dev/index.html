Fabrique development

Fabrique’s code and issue tracker are kept on GitHub. Pull requests (and other forms of code contribution) are very welcome!

Support tools

We currently use the GitHub issue tracker and code review system, but are thinking about switching to Gerrit. We don’t use any continuous integration at the moment, but would like to change that in the future.
